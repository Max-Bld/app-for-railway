This is the API

rootGet

Get Restaurants, Hotels or Point of Interests from DATAtourisme data base.

URL

https://b77c-2a02-8440-b217-ff76-4144-54ec-a87d-b1a1.ngrok-free.app/

HTTP Method

GET

Response Type

__get_200_response

Parameters

  • Query: query_element String

    Desired object's type.

  • Query: nom String

    Enter your name.

  • Query: ma_lat BigDecimal

    Personal GPS geolocation latitude.

  • Query: ma_lon BigDecimal

    Personal GPS geolocation longitude.

  • Query: file String

    If set to true, returns a json file as an attachment. Any other value will not return a json file.

  • Header: ngrok-skip-browser-warning String

    Skips ngrok warning page.
